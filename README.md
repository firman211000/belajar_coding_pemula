# Belajar Coding Pemula

Project latihan Python sederhana dengan menu utama, game tebak python, dan halaman warung.

## Menjalankan Program

```bash
python3 -m pip install -r requirements.txt
python3 main.py
```

Program ini membutuhkan MySQL lokal dengan database `warung_mini`.

## Struktur File

```text
main.py                 # Menu utama program
helpers.py              # Fungsi bantuan yang dipakai beberapa file
games/tebak_python.py   # Game tebak posisi python
tools/warung.py         # Halaman warung sederhana
```

## Menu

1. Permainan tebak python
2. Warung pintar
3. Keluar dari program
